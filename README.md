## ğŸŒŸ Overview
**Soulterra** is a 2D pixel-art **farming + survival + exploration + combat** game made with **Godot Engine (GDScript)**.  
Grow your farm, explore forests and dungeons, collect resources, fight monsters, and survive in a dynamic world with weather, day/night cycles, and atmospheric music.

This repository contains the source code for the current **BETA** version.
